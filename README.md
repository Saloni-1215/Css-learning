# 🎨 CSS Learning Repository

## 🚀 About
This repository contains my CSS learning journey from basic to advanced concepts.  
It is structured into multiple files, each focusing on a specific concept for better understanding and practice.

---

## 📚 Topics Covered

- CSS Basics (Selectors, Fonts)
- Colors & Backgrounds
- Box Model (Margin, Padding, Border)
- Flexbox (Layout Design)
- Responsive Design (Media Queries)
- Advanced CSS (Hover, Transitions)

---

## 📂 File Structure

- `01-basics.css` → Basic styling and selectors  
- `02-colors.css` → Colors and background properties  
- `03-box-model.css` → Margin, padding, border  
- `04-flexbox.css` → Layout using Flexbox  
- `05-responsive.css` → Media queries for responsiveness  
- `06-advanced.css` → Hover effects and transitions
